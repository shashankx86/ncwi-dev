# NCWI

### WebUI
frontend for ncwi 

### prerequisite 
- napi (ref to https://github.com/shashankx86/napi for setup instruction)

### Install

- Clone repo
  ```
  git clone https://github.com/shashankx86/ncwi-dev
  cd ncwi-dev
  ```
- Install deps
  ```
  bun install 
  ```
   or 
  ```
  npm install
  ```
- Build
  ```
  bun run build
  ```
  or
  ```
  npm run build
  ```
- Host
  moves build files from dist to public dir
